How to use

This is a RSS Feed Reader, it uses an XML format. It can be used to get the lastest updates on news, podcast, radio shows, etc. To try it out try entering http://rss.nzherald.co.nz/rss/xml/nzhtsrsscid_000000698.xml and click "Add" to add the url to the list, and then click "Goto RSS Feed" to see the top stories from the NZ Herald. Many of your favourite media outlets have RSS Feeds, try adding them in too.
